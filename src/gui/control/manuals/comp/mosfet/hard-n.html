Hard N

Shortcut: N

Hard N is n-channel MOSFET with Vgs(th) = +3.0V. It means that it would open if Vg - Vs >= +3.0V. And it would really do if it were totally ideal. But Terilog tries not to forget completely about unideal nature of components. That's why you should feel like Hard N 'does not like' positive source voltages: if source is positive and gate is stable - the drain will go positive (regardless to Vg).
