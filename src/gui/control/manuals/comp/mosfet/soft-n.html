Soft N

Shortcut: Shift+N

Soft N is n-channel MOSFET with Vgs(th) = +1.5V. It means that it would open if Vg - Vs >= +1.5V. And it would really do if it were totally ideal. But Terilog tries not to forget completely about unideal nature of components. That's why you should feel like Soft N 'does not like' positive source voltages: if source is positive and gate is stable - the drain will go positive (regardless to Vg).
