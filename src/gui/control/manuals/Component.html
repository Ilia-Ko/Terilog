Adding a component

Here is an instruction how to add a component to a circuit:

  1. Select a component in the menu (or submenu).
  2. The component appears over the circuit in 'flying' semi-transparent mode, following the mouse.
  3. Position the component by mouse (without clicking) and press left button in order to place it.

While positioning you can use these keys:

  • Escape or Delete - cancel component insertion
  • Open bracket [ - rotate left (counter clockwise)
  • Close bracket ] - rotate right (clockwise)
  • Quote ' - mirror horizontal
  • Backslash \ - mirror vertical

After you placed a component you can still do movement or transformation by right-clicking it.
